# Wine Experience Proposal

Landing estatica en HTML, CSS y JS para presentar la propuesta comercial de Impulzia para Wine Experience by Paz Cornu.

## Estado actual

- La pieza ya no esta planteada como landing publica del evento.
- El sitio funciona como documento comercial para ordenar y justificar una implementacion express ya ejecutada.
- La narrativa visible elimina CTAs de cierre y prioriza contexto, alcance, inversion, dependencias y condiciones.
- La modalidad comercial vigente es 50 por ciento al iniciar y 50 por ciento al terminar la campana.

## Stack

- HTML estatico en `index.html`
- CSS en `styles.css`
- JS liviano de animaciones y navegacion en `script.js`

## Estructura del repo

- `index.html`: propuesta comercial final visible
- `styles.css`: sistema visual Impulzia para esta pieza
- `script.js`: reveal on scroll y smooth scroll
- `propuesta-comercial-evento.md`: base comercial y condiciones fuente
- `landing-plan.md`: plan editorial y de narrativa de la propuesta
- `assets/images/`: logo SVG e imagenes generadas para la pieza

## Direccion de contenido

- Tono premium, editorial y consultivo.
- Framing comercial centrado en resolver un problema operativo contra reloj, no en vender una landing aislada.
- Fee principal: ARS 955.000.
- Pauta referencial: ARS 175.000 para 7 dias.
- Total estimado: ARS 1.130.000.

## Como abrir la pieza

Abrir `index.html` directamente en navegador. No requiere build ni dependencias.

## Notas de mantenimiento

- Si se cambia el framing comercial, alinear tambien `landing-plan.md` y `propuesta-comercial-evento.md`.
- Si se reemplazan assets, mantener nombres o actualizar las referencias en `index.html`.
- Evitar reintroducir CTAs de aprobacion mientras la negociacion siga abierta.
